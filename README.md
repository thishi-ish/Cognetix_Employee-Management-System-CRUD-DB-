# Employee Management System (SQLite)

Simple console-based employee manager using SQLite.

## Features
- Add, view, update, delete employees
- Data stored in `employees.db` (SQLite)
- Simple validation for email and salary

## Run
1. Install Python (3.7+).
2. Run:
   python employee_management.py

DB file will be created automatically.
